# Analysis of Value at Risk (VaR) Using the Exponentially Weighted Moving Average (EWMA) and Robust Exponentially Weighted Moving Average (Robust EWMA) Approaches: A Case Study of PT Bank Rakyat Indonesia (Persero) Tbk Stock
## Project Overview
Investasi saham merupakan aktivitas yang bertujuan memperoleh keuntungan di masa mendatang, namun tetap memiliki risiko kerugian yang perlu dikelola secara kuantitatif. Proyek ini bertujuan untuk membandingkan tiga metode estimasi risiko, yaitu Variance–Covariance, Exponentially Weighted Moving Average (EWMA), dan Robust EWMA, dalam mengukur Value at Risk (VaR) saham BBRI.JK (PT Bank Rakyat Indonesia (Persero) Tbk). Pengukuran VaR dilakukan dengan horizon waktu 1 hari dan eksposur awal sebesar Rp1 miliar. Hasil analisis menunjukkan bahwa nilai VaR meningkat seiring dengan kenaikan tingkat kepercayaan, sesuai dengan teori manajemen risiko. Pada tingkat kepercayaan 95%, estimasi potensi kerugian maksimum yang mungkin terjadi adalah:
- Rp28.770.000 (Variance–Covariance)
- Rp28.038.000 (EWMA)
- Rp25.812.000 (Robust EWMA)
  
Perbedaan metode menghasilkan estimasi VaR yang berbeda, sehingga pemilihan metode perlu disesuaikan dengan karakteristik data return saham. Proses backtesting menggunakan Kupiec Test menunjukkan bahwa ketiga metode valid dalam mengukur potensi kerugian saham BBRI.JK pada periode pengamatan. Proyek ini mengilustrasikan penerapan model volatilitas berbasis pembobotan eksponensial serta validasi statistik dalam manajemen risiko keuangan.
